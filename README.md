# E-Commerce Backend System

## Overview

This project implements a RESTful backend for an e-commerce platform using Java, Spring Boot, and MySQL.

## Features

* Product catalog management
* Customer management
* Order processing system
* Inventory tracking
* MySQL database integration

## Tech Stack

* Java
* Spring Boot
* MySQL
* REST APIs
* JDBC / JPA

## Database Tables

* users
* products
* categories
* orders
* order_items
* payments

## Project Status

Currently under development.
